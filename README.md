# Simple Project Manager

Это MVP веб-приложения для личного учета проектов и расходов. Код написан на чистом Node.js без сторонних зависимостей и использует TailwindCSS для оформления интерфейса.

Приложение позволяет добавлять проекты и расходы, настраивать базовые параметры и видеть расчёт чистой прибыли и вашей доли. В разделе "Финансы" отображается также небольшой график доходов по месяцам.
Теперь можно создавать задачи с описанием и дедлайном. Внутри проектов и задач отображаются таймеры до окончания, красные если осталось меньше суток. На главной выводятся уведомления о горящих дедлайнах. Также можно просмотреть историю действий в разделе "История".

## Запуск

1. Установите Node.js (версии 18+).
2. В терминале перейдите в папку проекта и запустите:

```bash
node api/server.js
```

3. Откройте браузер и перейдите по адресу `http://localhost:3000`.
   Сразу откроется список проектов, а сверху расположено меню со ссылками на остальные разделы.

Данные сохраняются в JSON-файлах в папке `data`. API доступен по маршрутам `/api/*` и может быть использован в дальнейшем для Telegram‑бота.

Интерфейс стилизован TailwindCSS, подключенным через CDN. На каждой странице есть кнопка "Тёмная тема" для переключения оформления на светлую или тёмную палитру.
Выбранная тема сохраняется в `localStorage` и применяется на всех страницах.

Дополнительно:

- В проектах можно редактировать задачи через кнопку ✏️ напротив каждой задачи.
- Файл `pages/requests.js` обеспечивает единый обработчик запросов и выводит ошибки API в консоль браузера.
- В админке теперь можно управлять списком клиентов для проектов.
- На странице "Отчёты" можно сформировать проектные и финансовые отчёты и скачать их в формате Word.
- Поля выбора (клиент, тип и т.д.) выглядят одинаково в светлой и тёмной темах.
- Раздел "История" хранит все действия с проектами и задачами.
